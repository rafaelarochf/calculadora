# Calculadora em Python

## Descrição

Este projeto consiste em uma calculadora simples desenvolvida em Python, com o objetivo de realizar operações matemáticas básicas por meio de um programa executado no terminal.

---

## Estrutura do projeto

O repositório contém os seguintes arquivos:

- **Calculadora_RafaelaFreitas.py**  
  Arquivo responsável pela implementação da calculadora em Python.

- **executar.sh**  
  Script em Shell utilizado para executar o programa Python.

- **README.md**  
  Arquivo de documentação do projeto.

---

## Como executar o arquivo executar.sh

Para executar a calculadora, siga os passos abaixo:

1. Certifique-se de que o Python esteja instalado em sua máquina.
2. No terminal, navegue até o diretório onde o projeto está localizado.
3. Conceda permissão de execução ao arquivo `executar.sh` com o comando:

chmod +x executar.sh

4. Execute o script utilizando o comando:

./executar.sh


Após a execução, o programa será iniciado no terminal.

---

## Explicação do código em Python

O arquivo **Calculadora_RafaelaFreitas.py** implementa uma calculadora que permite ao usuário realizar as seguintes operações matemáticas:

- Soma  
- Subtração  
- Multiplicação  
- Divisão  

O programa solicita ao usuário a escolha da operação desejada, recebe dois valores numéricos e exibe o resultado correspondente.
