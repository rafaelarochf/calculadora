# Calculadora 

Este repositório contém uma calculadora simples desenvolvida em Python, responsável por realizar operações matemáticas básicas como soma, subtração, multiplicação e divisão, além de um script em Shell (`executar.sh`) para facilitar a execução do código Python.

## Como executar

### Clonar o repositório
Primeiro, clone este repositório para a sua máquina:

git clone https://github.com/rafaelarochf/calculadora.git


### Navegar até o diretório do projeto
Após clonar o repositório, acesse a pasta onde ele foi salvo:


cd calculadora


### Verificar o script de execução
O arquivo `executar.sh` é responsável por executar o código Python da calculadora.

O conteúdo do script é:

#!/bin/bash

python Calculadora_RafaelaFreitas.py



### Tornar o script executável
Antes de executar o script, é necessário conceder permissão de execução:

chmod +x executar.sh


### Executar a calculadora
Após conceder as permissões, execute o script com o comando:

./executar.sh


A calculadora será iniciada no terminal.

---

## Explicação do código em Python

O arquivo `Calculadora_RafaelaFreitas.py` contém a implementação da calculadora em Python.

O programa solicita ao usuário a escolha da operação desejada e a entrada de dois valores numéricos, realizando em seguida o cálculo correspondente e exibindo o resultado no terminal.

As operações disponíveis são:

- Soma  
- Subtração  
- Multiplicação  
- Divisão
